## // About

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+what+this+is;%3E+why+it+exists" alt="About typing" />
</p>

An agentic system that audits Excel financial models across 12 dimensions and produces a single client-ready report - catching the errors humans miss and the ones LLM-built models routinely introduce. Already integrated into Claude Code as the `audit-general:*` skill family.

<a id="features"></a>

## // What's inside

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+features+%C2%B7+goals" alt="Features typing" />
</p>

- `▸` 12 specialist audit skills: Sentry, Logic, AI, Stylist, Efficiency, Lingo, Architect, Hyperlinks, VBA, Power Query, Navigator, Cartographer
- `▸` Orchestrator (`ag-audit-manager`) fans out, deduplicates findings, consolidates the report
- `▸` Client-ready audit report generator with walkthrough-first structure
- `▸` Mermaid dependency maps via the Cartographer agent
- `▸` Generalised CLI tools via `ag-generalizer` for reuse
